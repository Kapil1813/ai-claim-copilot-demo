# AI Healthcare Claim Denial Predictor

This project is a **real-time AI prototype** for predicting claim denials in healthcare.  
It simulates a **hospital revenue cycle management workflow** with dashboards, batch uploads, and AI explanations.

## Features

- Real-time claim input & denial prediction
- Batch claim upload & analysis
- Denial risk distribution charts
- Average denial risk by insurer & procedure
- Top denial reasons pie chart
- AI explanations for predicted denials

## Tech Stack

- Python
- Streamlit
- scikit-learn
- pandas
- OpenAI API (optional)
- Plotly & Seaborn for visualizations
